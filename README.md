== Utilización de los ejecutables ================================================================

Para ejecutar el programa, se debe correr primer el proceso Servidor.java, y luego el cliente
gráfico (que utiliza objetos clase Cliente) ejecutando TareaIIIRedes.java (este contiene un main
que crea la ventana).

En el cliente gráfico, se debe especificar la IP donde se esta corriendo el servidor (127.0.0.1 si
es local), y el puerto, que debe ser 8003 para conectarse con nuestro servidor (para modificar
este puerto, habría que editar el código fuente de Servidor.java).

Luego se debe mandar la consulta, seleccionando previamente el formato de respuesta.
